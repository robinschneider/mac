# mac
This repository is used to automate/backup tools used on my MacBook.

## Brew
### Install brew
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### Install applications
```bash
brew bundle --file=Brewfile
```
